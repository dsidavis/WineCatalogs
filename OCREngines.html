OCR Engines

There are 3 OCR engines we have considered, 1 open source (tesseract) and 2 commercial (Abbyy Reader and PDFPenPro).

We have processed all of the wine catalog images via PDFPenPro and are willing to reprocess specific images with different preprocessing.

tesseract
Interfaces from R and Python.
  • Rtesseract
  • pytesseract
  • tesseract
Abbyy
  • Online/Web API with support for processing a small number for free. You need to register and get authentication codes.
  • Can return the results in various formats: XML, DOCX, ...
  • Python code to submit an image to the Web API
  • R code to read the resulting XML documents from Abbyy.
PDFPenPro
PDFPenPro is an application on can purchase and process individual images or entire scanned documents. It does more than OCR.
